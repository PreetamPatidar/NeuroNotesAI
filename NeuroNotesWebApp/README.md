# NeuroNotes AI Web Dashboard

## Quick Start
1. Run your FastAPI backend on `localhost:8000`.
2. Open `index.html` in any browser.
3. Dashboard loads automatically from `/api/users/dashboard`.

## Features
- Modern responsive UI (CSS Grid, Material Design 3 inspired).
- Fetch API with error/loading states.
- Same JSON structure as Android app.
- Ctrl+R to refresh data.

## Shared Backend
- Web uses `localhost:8000` (browser).
- Android uses `10.0.2.2:8000` (emulator).
- Same endpoint and data model.

Fully self-contained - no build tools needed.
